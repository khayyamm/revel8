# Fast & Reckless Bank - Full Stack System

## Overview
A simple in-memory banking system built with Spring Boot backend and React frontend.

## Tech Stack
- Backend: Java 21, Spring Boot
- Frontend: React (Vite)
- Data storage: In-memory (no database)

## Features
- Create account
- Deposit money
- Withdraw money
- Transfer money between accounts
- View accounts
- View outgoing transfers

## Architecture
Frontend (React) → REST API → Backend (Spring Boot) → In-memory storage
